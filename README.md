# DashBot

DashBot is a real-time dashboard for chat bots. Also, it is a side project to practice my full-stack skills.

# Related Services and Libraries
- [LINE Messaging API](https://developers.line.biz/en/docs/messaging-api/overview/)
- [AWS Lambda](https://aws.amazon.com/tw/lambda/)
- [AWS EC2](https://aws.amazon.com/tw/ec2/)
- [Gin](https://github.com/gin-gonic/gin)
- [RethinkDB](https://rethinkdb.com/)
- [Svelte](https://svelte.dev/)
- [Tailwindcss](https://tailwindcss.com/)
